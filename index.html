Nivel Básico

=SUMA()

Suma todos los números en un rango de celdas.

=SUMA(A1:A10)

=PROMEDIO()

Calcula la media aritmética de un conjunto de valores.

=PROMEDIO(B2:B20)

=MAX() / =MIN()

Encuentra el valor máximo o mínimo en un rango.

=MAX(C1:C100)
=MIN(C1:C100)

=CONTAR()

Cuenta la cantidad de celdas con números en un rango.

=CONTAR(D1:D50)

Conceptos fundamentales:

  1. Introducción a la interfaz de Excel
  2. Formatos básicos de celdas (números, fechas, texto)
  3. 10 funciones esenciales para empezar
  4. Creación y formato de tablas básicas
  5. Gráficos simples (barras, líneas, pastel)
  6. Impresión y configuración de página
Tip profesional: Usa F2 para editar celdas rápidamente y Ctrl+Enter para confirmar cambios sin moverte de la celda.

Nivel Intermedio

=BUSCARV()

Búsqueda vertical en tablas (Excel tradicional).

=BUSCARV(valor, rango, columna, FALSO)

=X.BUSCAR()

Búsqueda moderna (versiones recientes de Excel).

=X.BUSCAR(valor, rango_busqueda, rango_resultado)

=SI()

Evaluaciones condicionales básicas.

=SI(condición, valor_si_verdadero, valor_si_falso)

=SI.CONJUNTO()

Múltiples condiciones en una sola función.

=SI.CONJUNTO(condición1, resultado1, condición2, resultado2, ...)

Temas cubiertos:

  • Validación de datos (listas desplegables, restricciones)
  • Formatos condicionales avanzados

Importante: Las funciones BUSCARV y X.BUSCAR requieren que los datos estén ordenados correctamente para funcionar.

Tabla Dinámica Ejemplo
¿Qué es una tabla dinámica?

Una tabla dinámica es una herramienta poderosa de Excel que te permite resumir, analizar, explorar y presentar datos. Puedes usarla para:

  • Resumir grandes conjuntos de datos
  • Analizar tendencias y patrones
  • Comparar diferentes variables
  • Crear informes dinámicos
Ejemplo de Macro VBA

Este es un ejemplo simple de una macro que formatea una tabla:

Sub FormatearTabla()
    ' Aplicar formato a los encabezados
    Range("A1:D1").Font.Bold = True
    Range("A1:D1").Interior.Color = RGB(146, 208, 80) ' Verde Excel
    
    ' Aplicar bordes a toda la tabla
    Range("A1:D100").Borders.LineStyle = xlContinuous
    
    ' Ajustar el ancho de las columnas automáticamente
    Columns("A:D").AutoFit
    
    ' Mostrar mensaje de finalización
    MsgBox "Tabla formateada correctamente", vbInformation
End Sub
Para usar macros, debes habilitar el editor de Visual Basic (Alt+F11) y habilitar las macros en las opciones de Excel.
Power Query Demo
¿Qué es Power Query?

Power Query es una herramienta de transformación de datos que te permite:

  • Importar datos de múltiples fuentes (Excel, CSV, bases de datos, web, etc.)
  • Limpiar y transformar datos de manera eficiente
  • Automatizar procesos de preparación de datos
  • Crear flujos de trabajo reproducibles

Power Query está integrado en Excel (desde 2016) como "Obtener y Transformar Datos".

Power Pivot y Modelo de Datos
Características principales:
  • Manejo de millones de filas de datos
  • Creación de relaciones entre tablas
  • Uso de DAX (Data Analysis Expressions)
  • Creación de medidas avanzadas
  • Análisis de datos complejos
Power Pivot no viene activado por defecto en Excel, debes habilitarlo en los complementos.
Dashboard Interactivo
Elementos de un buen dashboard:
  1. Visualizaciones claras: Gráficos adecuados al tipo de datos
  2. Interactividad: Filtros y segmentaciones de datos
  3. Jerarquía visual: Información más importante más destacada
  4. Consistencia: Mismo estilo y colores en todo el dashboard
  5. Performance: Optimizado para no ralentizar Excel

Un buen dashboard permite tomar decisiones rápidas basadas en datos.
